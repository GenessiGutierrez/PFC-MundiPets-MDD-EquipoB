# Figuras del informe

Contiene todas las imágenes referenciadas desde `Docs/informe.tex`: diagramas UML, capturas del proceso de generación, compilación, verificación y round-trip, además del código C# generado usado en los anexos.

Los archivos se nombran de forma descriptiva según la figura que representan en el informe (por ejemplo, `DiagramadeClases-GestionSolicitudAdopcion.png`, `cap_compilacion_exitosa.png`). No modificar los nombres, ya que están enlazados directamente desde el `.tex`.
