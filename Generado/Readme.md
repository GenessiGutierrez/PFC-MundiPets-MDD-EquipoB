## Notas importantes

- Este es el código **tal como lo produjo el generador**, sin correcciones manuales.
- El generador conserva literalmente los tipos escritos en el modelo, por lo que algunos (`date`, `boolean`) no son válidos en C# y requieren corrección manual (`DateTime`, `bool`) para compilar. El detalle de estas correcciones está documentado en el informe (Sección 8) y en las evidencias.
- El código constituye una **base estructural** (clases, campos y firmas de métodos); no incluye lógica de negocio, persistencia ni validaciones.
